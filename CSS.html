3. Colors in CSS

You can set colors using names, HEX codes, RGB, or HSL values.


h1 { color: red; }          /* color name */
h1 { color: #ff0000; }      /* HEX */
h1 { color: rgb(255, 0, 0);} /* RGB */
h1 { color: hsl(0, 100%, 50%);} /* HSL */

8. Selectors and Specificity

Selectors define which elements your CSS rules apply to. Understanding specificity helps you know which rule takes priority.


/* Element selector */
p { color: blue; }

/* Class selector */
.highlight { color: red; }

/* ID selector */
#main { color: green; }

/* Attribute selector */
[type="text"] { border: 1px solid gray; }

/* Pseudo-class */
a:hover { color: orange; }

Specificity hierarchy: ID > Class > Element

9. Grid Layout

CSS Grid makes complex page layouts easy to build and control. You can define rows, columns, and gaps between elements.


.grid-container {
  display: grid;
  grid-template-columns: 1fr 2fr;
  gap: 10px;
}

.grid-item {
  background-color: lightblue;
  padding: 20px;
  text-align: center;
}

10. CSS Variables

CSS variables (also known as custom properties) help you reuse values and keep your design consistent.


:root {
  --main-color: #3498db;
  --text-color: white;
}

button {
  background-color: var(--main-color);
  color: var(--text-color);
  padding: 10px 15px;
  border: none;
  border-radius: 5px;
}

  background-color: #2c80b4;
