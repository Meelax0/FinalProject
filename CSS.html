1. What is CSS?

<CSS (Cascading Style Sheets) is used to control the appearance of HTML elements. With CSS, you can change colors, fonts, layouts, and make your website responsive.>

2. CSS Syntax

Every CSS rule has three parts: a selector, a property, and a value. 


  selector {
    property: value;
  }

Example: This CSS makes all paragraphs blue:


  p {
    color: blue;
  }

3. Colors in CSS

You can set colors using names, HEX codes, RGB, or HSL values.


  h1 { color: red; }          /* color name */
  h1 { color: #ff0000; }      /* HEX */
  h1 { color: rgb(255, 0, 0);} /* RGB */
  h1 { color: hsl(0, 100%, 50%);} /* HSL */

4. Backgrounds

The background-color property changes the background of elements.


  body {
    background-color: green;
  }

5. The Box Model

All elements in CSS use the box model: content → padding → border → margin.

6. Fonts and Text

CSS lets you control font families, sizes, and styles.


  p {
    font-family: Arial, sans-serif;
    font-size: 16px;
    font-weight: bold;
    text-align: center;
  }

7. Layout with Flexbox

Flexbox makes it easy to align and distribute space in a container.


  .container {
    display: flex;
    justify-content: center;
    align-items: center;
  }
