1. Basic Structure of an HTML Page

An HTML document always starts with a <!DOCTYPE> declaration, followed by the <html> tag. Inside it, we have a <head> (page info) and <body> (page content).

Example Code:

                        <!DOCTYPE html>
                        <html>
                        <head>
                            <title>My First Page</title>
                        </head>
                        <body>
                                <h1>Hello World!</h1>
                                <p>This is my first HTML page.</p>
                        </body>
                        </html>

Result:

Hello World!

This is my first HTML page.

2. Headings & Paragraphs

Headings define titles and subtitles. <h1> is the biggest, <h6> is the smallest. Paragraphs are written with <p>.

        <h1>This is a main heading</h1>
        <h2>This is a sub-heading</h2>
        <p>This is a paragraph of text.</p>

4. videos & Audios

HTML allows you to embed videos and audio files directly into your web pages.
You can use the <video> tag for videos and the <audio> tag for sound.
Both tags can include attributes like controls, autoplay, and loop

                        <!-- Video Example -->
        <video width="320" height="240" controls>
          <source src="videos/sample-video.mp4" type="video/mp4">
          Your browser does not support the video tag.
        </video>
        
                        <!-- Audio Example -->
        <audio controls>
          <source src="audio/sample-audio.mp3" type="audio/mpeg">
          Your browser does not support the audio element.
        </audio>

5. Lists

Lists can be unordered (<ul>, with bullets) or ordered (<ol>, with numbers). Items go inside <li> tags.

        <ul>
          <li>Apples</li>
          <li>Bananas</li>
          <li>Cherries</li>
        </ul>
        
        <ol>
          <li>First</li>
          <li>Second</li>
          <li>Third</li>
        </ol>

Result:

  • Apples
  • Bananas
  • Cherries
